📊 Fonctionnement technique

Logique de mapping :
1. L'utilisateur clique sur un nom d'entreprise
2. Le système vérifie s'il existe un mapping
3. Si oui → utilise le nom mappé pour la recherche
4. Si non → utilise le nom original
5. Recherche via l'API /api/companies/search
6. Si trouvé → redirection vers /companies/{id}
7. Si non trouvé → redirection vers /companies?q=nom
