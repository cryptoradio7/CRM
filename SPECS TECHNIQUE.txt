ARCHITECTURE
Hébergement local
Frontend: 
 React -> interface user -> fiches prospects, listes, cartes localisation, sélection multiple
 TypeScript -> détecte erreurs, lisibilité code, 
Backend: 
 Node.js -> exécute javascript côté serveur, gère logique métier, back end application, requêtes DB
 Express -> API, FO qui communique avec serveur
 TypeScript 
Base de données: PostgreSQL
